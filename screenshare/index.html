### Cobrowse - Very fast, low bandwidth - Limited to the browser content; cannot capture Quicken desktop application - AKA "DOM Diffing"
![cobrowse](images/cobrowse.png)
### Screen Capture (WebRTC) - Works in Web and desktop application contexts - Requires higher bandwidth (about 0.5 - 1mbps) - Additional installs when the customer is in... - Quicken Desktop App: **None** - Firefox: **None** - Chrome: Lightweight extension (one click) \*\* No Java or Flash installation \*\*
